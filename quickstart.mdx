# Quickstart

This guide covers two paths: using the REST API directly, or using the `@billy/sdk` TypeScript package.

## Prerequisites

- A Billy account and a project created in the dashboard
- A Stripe account with at least one product and price configured
- Node.js 18 or later (for SDK usage)

## Option 1: REST API

All endpoints accept JSON and require your API key.

```bash
curl -X POST https://api.usebilly.dev/api/customers \
  -H "Authorization: API Key <your_api_key>" \
  -H "Content-Type: application/json" \
  -d '{
    "projectId": "proj_abc123",
    "externalUserId": "user_42",
    "email": "jane@example.com",
    "name": "Jane Smith"
  }'
```

A `201` response includes the new customer object under `data`.

## Option 2: TypeScript SDK

### Install

<CodeGroup>
```bash npm
npm install @billy/sdk
```
```bash yarn
yarn add @billy/sdk
```
```bash pnpm
pnpm add @billy/sdk
```
</CodeGroup>

### Initialize

```typescript
import { Billy } from '@billy/sdk';

const billy = new Billy('<your_api_key>');
```

Store your API key in an environment variable — never hardcode it.

```typescript
// .env
BILLY_API_KEY=bly_live_...

// usage
const billy = new Billy(process.env.BILLY_API_KEY!);
```

### Create your first customer

```typescript
const customer = await billy.customers.create({
  projectId: 'proj_abc123',
  externalUserId: 'user_42',
  email: 'jane@example.com',
  name: 'Jane Smith',
});

console.log(customer.data.id); // cus_...
```

### Create an enterprise subscription

```typescript
const subscription = await billy.subscriptions.createEnterprise({
  customerId: customer.data.id,
  priceId: 'price_abc123',
  seats: 5,
});
```

<Note>
  Standard subscriptions are created through the customer portal or checkout flow. The `/api/subscriptions/enterprise` endpoint is reserved for programmatic, server-side provisioning of enterprise accounts.
</Note>

### Add tokens

If your project uses usage-based tokens, grant an initial balance after the subscription is active.

```typescript
await billy.tokens.add({
  customerId: customer.data.id,
  amount: 10000,
  description: 'Initial grant on signup',
});
```
